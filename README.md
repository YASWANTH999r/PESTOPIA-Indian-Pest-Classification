# 🐛 Pestopia: Indian Pest Classification

> Deep Learning project for multiclass **Indian pest classification** (132 classes, 55,865 images) using **VGG16**, **ResNet50**, and a custom **CNN**, with **pesticide recommendation** via **K-Means clustering**.

---

## 📖 Project Overview
This project focuses on pest detection and pesticide recommendation using deep learning and clustering.  
- ✅ 132 pest classes  
- ✅ Dataset size: 55,865 images  
- ✅ Pretrained Models: VGG16, ResNet50  
- ✅ Custom CNN architecture  
- ✅ Pesticide recommendation with K-Means  
- ✅ Achieved **92% accuracy** 🎯  

Dataset: [Pestopia on Kaggle](https://www.kaggle.com/datasets/shruthisindhura/pestopia)
